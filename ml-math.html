📊 機率分佈（核心考點）
五大機率分佈：特性、公式、機器學習應用場景
常態分佈 N(μ, σ²)
鐘型對稱，均值=中位數=眾數。
68-95-99.7 法則：
μ±1σ≈68%，μ±2σ≈95%，μ±3σ≈99.7%
中央極限定理：大樣本均值→常態
▶ 誤差分佈假設、初始化權重
伯努利分佈 Bernoulli(p)
P(X=1)=p, P(X=0)=1-p（二元結果）
均值=p，方差=p(1-p)
邏輯迴歸假設目標服從伯努利
最大化伯努利似然 ↔ 最小化 BCE
▶ 考點：邏輯迴歸假設 Bernoulli
卡方分佈 χ²(k)
k 個獨立標準常態平方和，右偏正值
均值=k，方差=2k
用於特徵獨立性檢定（SelectKBest）
自由度越大越接近常態
▶ 考點：特徵選擇、獨立性檢定

🔮 貝氏定理（Bayes' Theorem）
條件機率更新：先驗 × 似然 → 後驗（機器學習最核心公式之一）
// 貝氏定理核心公式
P(A|B) = P(B|A) × P(A) / P(B)

// 各項意義
P(A|B) → 後驗機率（觀察到 B 後 A 的機率）
P(B|A) → 似然函數（給定 A 時 B 發生的機率）
P(A) → 先驗機率（觀察前對 A 的初始估計）
P(B) → 邊際機率（正規化常數）

// 樸素貝氏（假設特徵條件獨立）
P(C|x₁,...xₙ) ∝ P(C) × ∏ P(xᵢ|C)
醫療診斷情境（高頻考點）
疾病盛行率 1%（先驗低），檢測陽性敏感度 99%，假陽性率 5%。
陽性但實際患病率 ≈ 17%！
貝氏更新：先驗機率低→即使似然高，後驗仍可能出乎意料低
▶ 考點：先驗低→後驗意外低
最大似然估計（MLE）
最大化 P(data|θ)，找讓資料出現機率最大的參數 θ。
等效：最小化負對數似然（NLL）
邏輯迴歸訓練 = 最大化伯努利似然 = 最小化交叉熵
▶ 考點：MLE ↔ 最小化交叉熵
貝氏最佳化（超參數調校）
用高斯過程建立目標函數代理模型，以後驗引導超參數搜索。
比 Grid Search 更高效：同等預算找到更好超參數。
採集函數（Acquisition Function）決定下一個試驗點
▶ 考點：超參數調校策略
🔢 特徵值分解與 SVD 奇異值分解
矩陣分解是 PCA 降維、推薦系統、NLP 語義分析的數學基礎
// 特徵值分解（方陣）
Σv = λv → Σ = Q Λ Qᵀ
// λ=特徵值，v=特徵向量，Q=特徵向量矩陣，Λ=對角特徵值矩陣

// PCA：對協方差矩陣做特徵值分解
PCA: Cov(X) → 特徵值分解 → 選前 k 大特徵向量（主成分）

// SVD 奇異值分解（任意矩陣）
X = U Σ Vᵀ
// U=左奇異向量(m×m)，Σ=奇異值對角矩陣，V=右奇異向量(n×n)
Truncated SVD: 保留前 k 個奇異值 → 低秩近似，重建誤差最小
LSA: 文本 TF-IDF 矩陣做 SVD → 語義主題
特徵值的幾何意義
特徵向量：矩陣變換後方向不變的向量
特徵值 λ：變換後的縮放比例
λ 越大 → 資料在此方向的方差越大
PCA 選 λ 最大的 k 個特徵向量
▶ 考點：大特徵值=大方差方向
SVD vs 特徵值分解
特徵值分解：僅適用方陣（n×n）
SVD：適用任意矩陣（m×n），更通用
全連接層參數壓縮 → SVD 低秩分解
推薦系統協同濾波 → 矩陣分解（MF）
▶ 考點：SVD 比特徵值分解更通用
LSA（潛在語義分析）
詞-文件矩陣（TF-IDF）做 Truncated SVD
→ 找出隱藏語義主題
奇異值代表主題的重要性
U→文件-主題，V→詞-主題
解決同義詞（多詞同義）問題
▶ 考點：LSA = TF-IDF + SVD
⛰️ 3D 損失曲面與梯度下降動畫
SGD vs Adam 在 3D 損失曲面上的收斂軌跡（可拖曳旋轉）
⛰️ 損失曲面 3D · 拖曳旋轉
// 梯度下降三大變體（核心考點）
BGD ：θ ← θ - η·∇L(全部資料) → 穩定但慢，每步用全資料
SGD ：θ ← θ - η·∇L(單一樣本) → 快但震盪，可跳出局部最小
Mini-batch：θ ← θ - η·∇L(batch B) → 實務首選（batch=32/64）

// Adam = Momentum + RMSProp（自適應學習率）
m_t = β₁·m_{t-1} + (1-β₁)·g_t // 一階動量（β₁=0.9）
v_t = β₂·v_{t-1} + (1-β₂)·g_t² // 二階動量（β₂=0.999）
θ ← θ - η · m̂_t / (√v̂_t + ε) // 偏差修正後更新
⚡ 最佳化器比較（Adam / Adagrad / Momentum）
各最佳化器特性、適用場景與考試重點
最佳化器核心機制優點缺點適用場景
SGD固定學習率更新記憶體小、簡單震盪大、需調學習率大規模、簡單任務
Momentum加入動量項 β·v_{t-1}加速收斂、減少震盪可能越過極小值損失曲面多鞍點
Adagrad累積梯度平方調整 η稀疏特徵自動大步長學習率單調遞減→後期停止更新稀疏資料（NLP 詞嵌入）
RMSProp指數衰減移動平均梯度平方解決 Adagrad 遞減問題需調衰減率 ρRNN 時序模型
AdamMomentum + RMSProp + 偏差修正收斂快、自適應、魯棒泛化有時不如 SGD深度學習實務首選
Adam 預設超參數
β₁ = 0.9（動量衰減率）
β₂ = 0.999（RMSProp 衰減率）
ε = 1e-8（防止除以零）
η（學習率）= 1e-3 或 3e-4
偏差修正：m̂_t = m_t/(1-β₁ᵗ)
▶ 考點：β₁=0.9, β₂=0.999
Adagrad 稀疏資料優勢
累積梯度平方：G_t += g_t²
更新：θ -= η/√(G_t+ε) · g_t
稀疏資料：少更新的參數 G_t 小→學習率大→充分更新稀有詞嵌入
缺點：G_t 只增不減→後期 η≈0
▶ 考點：Adagrad→稀疏資料/NLP
學習率衰減策略
Step Decay：每 N epoch 乘以 γ
Cosine Annealing：餘弦曲線衰減
Warm-up + Decay：先升後降（Transformer）
學習率過大 → 震盪不收斂
學習率過小 → 收斂太慢
▶ 考點：Warm-up → Transformer 常用
⚠️ 梯度消失與梯度爆炸
深度神經網路訓練最大挑戰：反向傳播中梯度異常（必考）
梯度消失（Vanishing Gradient）
梯度反向傳播逐層相乘 → 趨近 0
原因：Sigmoid/Tanh 飽和區導數<<1
後果：淺層權重幾乎不更新
解法：ReLU、BatchNorm、ResNet 殘差連結、LSTM 門控
▶ 考點：解法 → ReLU + ResNet
梯度爆炸（Exploding Gradient）
梯度逐層相乘 → 指數級增大 → NaN
原因：權重初始化過大、層數過深
後果：損失 NaN，模型發散
解法：梯度裁剪（Gradient Clipping）
clip_grad_norm_(params, max_norm=1.0)
▶ 考點：梯度裁剪解決爆炸
解決方案總覽
消失：ReLU（非飽和）、BatchNorm（標準化中間層）、殘差連結（梯度高速公路）、LSTM/GRU（記憶閘門）

爆炸：梯度裁剪 clip_grad_norm_、Xavier/He 初始化、L2 正則化
▶ 消失→ReLU/BN；爆炸→裁剪
📌 iPAS 必記：梯度消失→ReLU（正區間導數=1，不飽和）；梯度爆炸→梯度裁剪（clip max_norm）；BatchNorm 讓每層輸出標準化 → 兩者都能改善。RNN 最容易爆炸（時序反向傳播多步相乘）。

常態 vs 指數 vs 均勻分佈 — PDF 曲線比較
常態分佈 N(0,1)

PDF: f(x)=λe^(-λx)，均值=1/λ

▶ 無記憶性是核心考點
均勻分佈 U(a,b)
每個值等機率，PDF=1/(b-a)
Xavier 初始化：U(-√6/√(n_in+n_out), +√6/√(n_in+n_out))
超參數隨機搜索的基礎分佈
均值=(a+b)/2，方差=(b-a)²/12
▶ Xavier 初始化用均勻分佈

NumPy — 特徵值分解與 SVD
import numpy as np

# 特徵值分解（方陣）
A = np.array([[4, 2], [1, 3]])
eigenvalues, eigenvectors = np.linalg.eig(A) # λ, v
# eigenvalues = [5., 2.] → 主成分方向的方差大小

# SVD 分解（任意矩陣）
U, S, Vt = np.linalg.svd(A, full_matrices=False) # X=UΣVᵀ
# S = 奇異值（降序排列）
k = 1 # 保留 k 個奇異值→低秩近似
A_approx = U[:, :k] @ np.diag(S[:k]) @ Vt[:k, :]

# PCA 手動實作
X_c = X - X.mean(axis=0) # 中心化
cov = np.cov(X_c.T)
vals, vecs = np.linalg.eigh(cov) # eigh：對稱矩陣，更穩定
X_pca = X_c @ vecs[:, -k:] # 選前 k 大特徵向量
np.linalg.eig vs eigh
eig：通用矩陣，可回傳複數特徵值
eigh：對稱/Hermitian 矩陣，回傳實數，更穩定數值
協方差矩陣是對稱矩陣 → 用 eigh 更準確
▶ 協方差矩陣用 eigh
np.linalg.svd 參數
full_matrices=True：完整 U(m×m), Vt(n×n)
full_matrices=False：精簡版，節省記憶體
奇異值 S 已按降序排列
低秩近似只需前 k 個奇異值
▶ full_matrices=False 節省記憶
解釋方差比例
截斷 SVD 最小化 Frobenius 範數誤差：
‖A - Aₖ‖² = Σ(i>k) σᵢ²
解釋比例 = Σσᵢ²(前k) / Σσᵢ²(全部)
類比 PCA 解釋累積方差比例
▶ 解釋比例 = 累積奇異值平方比

.cdf(x)：累積分佈 P(X≤x)

梯度下降與 Adam 手動實作（NumPy）
import numpy as np

# Mini-Batch SGD（線性迴歸示例）
theta = np.zeros(3) # 初始化參數
lr, batch = 0.01, 32
for epoch in range(100):
  idx = np.random.permutation(100) # 打亂索引
  for i in range(0, 100, batch):
    Xb, yb = X[idx[i:i+batch]], y[idx[i:i+batch]]
    grad = Xb.T @ (Xb @ theta - yb) / batch # MSE 梯度
    theta -= lr * grad

# Adam 最佳化器手動實作
m, v, t = 0, 0, 0 # 一階動量、二階動量、時間步
b1, b2, eps = 0.9, 0.999, 1e-8
t += 1
m = b1*m + (1-b1)*grad # 一階動量
v = b2*v + (1-b2)*grad**2 # 二階動量
m_hat = m/(1-b1**t) # 偏差修正
v_hat = v/(1-b2**t) # 偏差修正
theta -= lr * m_hat / (np.sqrt(v_hat) + eps)

# 梯度裁剪（防止爆炸）
norm = np.linalg.norm(grad)
if norm > 1.0: grad *= 1.0 / norm # 裁剪至 max_norm=1.0
批次大小影響
batch=1（SGD）：噪聲大，可跳出局部最小
batch=32/64：Mini-batch，速度與穩定性平衡
batch=全部（BGD）：精確梯度，記憶體需求大
實務 batch=32 或 64 最常見
▶ 實務 batch=32 或 64
偏差修正的必要性
t=1 時，m=0.1·grad（初始化為 0 帶來偏差）
m_hat = m/(1-0.9¹) = grad（修正後正確）
前幾步不修正→更新量偏小→初始收斂慢
t 越大，偏差修正因子趨近 1
▶ 偏差修正解決初始步偏小
